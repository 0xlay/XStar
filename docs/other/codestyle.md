# XStar code style

### Tabs vs Spaces
`We use 4 spaces`

### Naming style
| Code | Style |
| --- | ----------- |
| variable | `camelCase`|
| constant/macros | `UPPER_CASE_SNAKE_CASE` |
| namespace | `lowercase` |
| struct and class | `PascalCase` |
| method | `camelCase` |
| function | `PascalCase` |

| Code | Style |
| --- | ----------- |
| file name | `lowercase` or `snake_case`|
| folder name | `lowercase` or `snake_case`|
